# caproject

This project was bootstrapped with Vite, React, and TypeScript. It includes Tailwind CSS, shadcn/ui configuration, ESLint, Prettier, Vitest, and Husky.

## Getting Started

Install dependencies with `pnpm install` and start the development server with `pnpm dev`.

## Available Scripts

- `pnpm dev` - start the development server
- `pnpm build` - build for production
- `pnpm preview` - preview the production build
- `pnpm lint` - run ESLint
- `pnpm format` - run Prettier
- `pnpm test` - run unit tests with Vitest

Husky runs `pnpm lint` and `pnpm test` on every `git push`.

Configure shadcn/ui components via `components.json`.
